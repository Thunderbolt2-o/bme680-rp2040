# ** BME680-RP2040 **

This repository consits of example for interfacing _Rasberry pi pico board(RP2040)_ with _DFRobot BME680_ using _I2C_ interface. BME680 driver has been used to get all the functionality of the sensor. 

### ** Pin connections **

| PICO PIN | BME680 PIN |
| ------------- | ------------- |
| 3.3V  | VCC  |
| GND   | GND  |
| SDA   | 4    |
| SDA   | 5    |
